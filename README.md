# Bosco3D Lite

Segnapunti semplice per percorsi 3D di tiro con l'arco.

## Funzioni
- 24 bersagli fissi
- 2 frecce per bersaglio
- Punteggi 11, 10, 8, 5, M
- Totale in tempo reale
- Annulla ultimo colpo
- Annulla allenamento
- Salvataggio automatico
- Riprendi allenamento
- PWA offline

## GitHub Pages
Carica questi file in un repository, poi abilita Settings > Pages > Deploy from branch > main / root.
